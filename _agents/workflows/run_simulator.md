# Start the Flask Backend and React Frontend for the Booth Simulator

This workflow will start both the Python backend and the React frontend.

1. Install backend dependencies and start the Flask server
// turbo

```bash
cd C:\Users\itska\Desktop\booth-algorithm-simulator-main\booth-algorithm-simulator-main\QT_HACK-main
pip install -r requirements.txt
python app.py
```

2. Open a new terminal, install frontend dependencies, and start Vite
// turbo

```bash
cd C:\Users\itska\Desktop\booth-algorithm-simulator-main\booth-algorithm-simulator-main\frontend
npm install
npm run dev
```
